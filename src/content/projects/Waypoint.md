## Task Description

As part of a team project, we were tasked with creating a software product that either educates, or meets one of the UN Sustainable development goals. For ours, we created an AI powered travel itinerary planner. Meeting the goal of Decent work and Economic Growth, aiming to highlight lesser known attractions and / or prioritising smaller local businesses over large chains and corporations.

## Languages / Tools

- Python Flask for the backend v3.9.13
- We utilised Google Maps API to gather the attractions and places to visit once the user types in their desired location. Which is added to Mongo DB where the plans are stored.
- Gemini API to provide a summary of the location the user chooses, as well as a short description of each attraction.
- MySQL to store user data such, and Mongo DB to store user generated plans.
- Docker
- Redis v3.0.504
