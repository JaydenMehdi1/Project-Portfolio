**Banking App Project**



- This is a simple bank app made in Java using Jframes while queering a MySQL database to store information such as user data including:
    - User ID
    - Username
    - Password
    - Balance

As well as storing transaction data including:

- Transaction type (deposit, withdraw, or transfer)
- Transaction amount
- Transaction date
- user id



**Functionality Includes:**

- Registering for an account
- Logging into an account
- Deposits
- Withdraw
- Transfer to another user
- Logout



To run the project you should run the AppLauncher.java file
