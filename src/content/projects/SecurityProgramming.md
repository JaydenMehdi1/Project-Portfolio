## Task Description

Developed a basic blog app, allowing users to create, edit, delete posts. As part of a security module at University. On top of the general functions of the application, like creating / editing / deleting posts, registration / login etc. We had to implement.
• Implemented password hashing, encryption, 2FA, captcha, limited login attempts, rate limiting, error handling, validation, role based access control
